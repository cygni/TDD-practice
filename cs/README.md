
# Tdd Practice in chsarp

## To run

In the Src folder:

The first argument is what number to convert to roman.

```
> dotnet run 16
> 16 in roman is: XVI
```

## To run the tests

In the Test folder:

`dotnet test`

## To run the mutation tests

in the Test folder:

1. `dotnet tool restore`
2. `dotnet stryker`
